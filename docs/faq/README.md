### 无法播放

::: details 打开后白屏或一直转圈
请先确认设备`android`版本，一般低`android`版本不支持播放，此时需要下载`X5`内核版本，`X5`内核版本最高支持 `Android 13`，
你可以点击 菜单 -> 设置 -> `WebView`调试，然后查看`X5内核版本是否存在`，如果不存在，点击 `尝试安装X5内核`，安装后重启App。
最后，确认你是否下载对了版本，老旧设备一般是32位的，请下载32位版本。如果依然存在请在公众号给我发消息反馈。
:::

::: details 安装X5内核后直接闪退
确认你是否下载对了版本，老旧设备不支持64位版本。
:::

::: details 安装了还是没反应、没用
这时候需要手动下载安装 [AndroidSystemWebView](https://www.apkmirror.com/apk/google-inc/android-system-webview/) 下载好之后进行安装即可

这里提供以下下载地址:

### 安装

::: details 电视安装
USB数据线连接电视和手机(或U盘)，打开手机的通知，将仅充电改为文件传输，然后把安装包复制到电视，电视找到相关USB存储，安装apk
通过无线方式安装，需要 android 11 以上，打开电视 adb调试(如何开启请百度)，使用甲壳虫adb助手连接电视后安装
具体请百度或问AI，这里不进行详细描述

**安装事项**

有些电视限制安装第三方App，需要使用 adb 禁用软件包安装程序

tcl电视举例（其他品牌电视请问AI或百度）：需要电视开启adb，进入 设置 -> 关于 -> 依次按下 上下左右 ，之后会显示扩展设置， 找到开启adb开关，然后使用 <https://www.jiakechong.net/> 甲壳虫adb助手 连接到电视， 连接之前需要获取电视在网络中的ip，打开电视网络信息会有ipv4 地址，这个就是。然后用甲壳虫找到应用管理， 禁用软件安装包程序就可以安装app了。 安装完成之后再把软件安装包程序启用。
:::

### 开机自启

::: details 电视开机自启

1. 查找电视的一些设置、App，看看有没有应用自启动相关的设置，对本应用开启自启动。可百度或 Ai 搜索自启动相关文章
2. 确保 App 已经打开过，并允许了悬浮窗权限
3. 设置默认的桌面，一般按 Home键(主页键) 会提示设置默认桌面，可以设置为本App并选择始终，或查找电视相关设置和App设置默认桌面
4. 不保证所有电视都支持，仅在原生系统虚拟机 Android 12 测试通过

如果上面的方法不奏效，可以尝试以下方法：

1. 进入电视的工厂模式
2. 开启 USB 调试
3. 使用无线调试或数据线连接电视和电脑(手机-需要安装终端)电脑安装adb 手机可以安装MT管理器，自带终端
4. 进入终端，输入以下命令

```shell
# 无线连接 adb connect ip:端口

adb devices # 查看设备是否连接成功
adb shell # 进入管理员模式
cmd package set-home-activity "xyz.jdynb.tv/.MainActivity" # 设置默认的启动项
reboot # 重启设备
```

> 注意：定制系统可能无效

:::
